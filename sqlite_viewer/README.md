# SQLite Viewer

## Created By CommandO

### Installation

pip install -r requirements.txt

### Usage

python3 main.py

### Commands

open <path>     - Open SQLite database
show            - Show all tables
view <table>    - View table data
query <sql>     - Run SQL query
export <table> <file.csv> - Export to CSV
exportj <table> <file.json> - Export to JSON
close           - Close database
help            - Show help
exit            - Exit

### Requirements

Python 3.6+
tabulate
colorama
